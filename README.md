# Google Business Profile Manager

A local-first, AI-augmented Flutter application for managing Google Business Profiles, with responsive mobile/web desktop flows and a staging engine for bulk catalog operations.

## Architecture highlights

- Local-first storage using Drift (SQLite)
- Riverpod for state management and multi-tenant business selection
- go_router for responsive shell navigation
- AI-assisted drafting and local pricing estimation hooks
- Mock data seeding for real estate and tourism businesses
